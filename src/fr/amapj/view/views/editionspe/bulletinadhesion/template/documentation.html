Ce document fournit laliste des champs utilisables pour la génération des bulletins d'adhesion

Les informations de l'amapien

$amapien.nom : donne le nom de l'amapien

$amapien.prenom : donne le prénom de l'amapien

$amapien.email : donne l'email de l'amapien 

$amapien.numTel1 : donne le numéro de téléphone 1

$amapien.numTel2 : donne le numéro de téléphone 2

$amapien.libAdr1 :  donne le libellé de l'adresse

$amapien.codePostal : donne le code postal

$amapien.ville : donne la ville de l'amapien 

Les informations de l'adhesion

$adhesion.nomPeriode : donne le nom de la période de cotisation, 2016 par exemple 

$adhesion.dateDebut : donne la date de début de la période

$adhesion.dateFin : donne la date de fin de la période

$adhesion.montantAdhesion : donne le montant de l'adhésion renseigné par l'amapien

$adhesion.montantMini : donne le montant mini

$adhesion.montantConseille : donne le montant conseillé

$adhesion.dateDebutInscription : donne la date de début des inscriptions

$adhesion.dateFinInscription : donne la date de fin des inscriptions

$adhesion.textPaiement : donne le texte affiché pour les instructions de paiements

$adhesion.libCheque : donne l'ordre du chèque

$adhesion.dateRemiseCheque : donne la date de remise des chèques

$adhesion.dateAdhesion : donne la date à laquette l'amapien a validé son adhésion 

Les informations de date

$date.impression : donne la date de génération du document par le logiciel 

Les informations de l'AMAP

$amap.nom : donne le nom de l'AMAP

$amap.ville : donne la ville de l'AMAP
